## Installation

```bash
$ npm install
```

## Configuration

```
PORT=3001
JWT_SECRET=something-secret-like-this123
DB_USER=username
DB_PASS=password
DB_DOMAIN=cluster-name.something.mongodb.net
DB_NAME=database-name
IGDB_CLIENT_ID=client-id
IGDB_CLIENT_SECRET=client-secret
IGDB_ACCESS_TOKEN=access-token

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
